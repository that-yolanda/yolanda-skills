# yo-self-review

每日/每周复盘技能。分析你的日志与时间分配，识别高价值产出、低价值消耗、判断偏差、可沉淀资产——训练判断力，而非总结流水账。

## 何时用

- 你说「今日复盘」「每日复盘」「分析今天日志」→ 每日模式
- 你说「周复盘」「每周复盘」「分析本周日志」→ 每周模式

## 安装与首次配置

依赖 `NOTES_DIR`（笔记/复盘根目录）。对 agent 说「帮我配置 yo-self-review」引导设置。

## 它会做什么

读取当日/当周日志 → 提炼判断（不复述）→ 输出「今日洞察 / 本周洞察」追加到日志；其中的决策修正与可沉淀资产交给 yo-wiki-atom 录入，状态变化交给 yo-whoami 更新画像。
