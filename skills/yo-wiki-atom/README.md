# yo-wiki-atom

从你的输入中提取知识原子并写入原子库。当你分享了值得沉淀的信息、提供了链接或文档、想记录洞察时使用。

## 何时用

- 你说「记录一下这个」「存到知识库」
- 你分享了有长期价值的经验/链接/观点

## 安装与首次配置

依赖 `WIKI_DIR` 与可执行的 `yo wiki` 命令。仓库提供 `examples/yo.mjs` 作为读写 `atoms.jsonl` 的示例实现；对 agent 说「帮我配置 yo-wiki-atom」引导完成。

## 它会做什么

判断价值（「下个月还能不能用上」）→ 提取独立知识点 → 按类型归类 → `yo wiki add` 写入；高价值资料整理进资料库。
