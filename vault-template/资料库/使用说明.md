# 资料库使用说明

本文指导当有价值可复用资产需要保存到该目录时需要遵循的规范。

## 文件夹&文件整理规范
- 命名规范：文件夹文件命名遵循 [文件&目录管理规范](../我的画像/我的偏好.md#文件&目录管理规范)
- 文档格式：文档保存后检查内部的是否存在排版问题，如果有则修正，例如 排版错乱，错误的结构，违反markdown的语法的问题
- 图片路径：文档中如果有下载的图片，统一保存在文档同层级路径下，以`资源-文档标题` 命名文件夹，并修改文档中的内部链接
- frontmatter：保存后的 markdown 文档需要添加以下几个frontmatter，date：文档的发布日期，url：文档原文url，author：作者，id：原子信息id，tags：标签（具体定义参考[原子库字段说明](../原子库/使用说明.md#字段说明)）
- 资料库整理：定期整理资料库的文档，按照专题来管理，方便查找
